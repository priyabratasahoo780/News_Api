NeonNews Pro is a premium, high-performance news aggregator that delivers real-time global news with a stunning, modern aesthetic. Featuring a full-width immersive layout, interactive 3D elements, and a robust mobile-first drawer system.

![App Preview](https://images.unsplash.com/photo-1504711434969-e33886168f5c?auto=format&fit=crop&w=1200&q=80)

---

## ✨ Key Features

- 🚀 **Real-time News**: Powered by the NewsData.io API for the latest stories across categories.
- 🎨 **Premium UI/UX**: A glassmorphic editorial design with a dark/light mode toggle.
- 📱 **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile with a dedicated hamburger navigation drawer.
- 🧊 **Interactive Elements**: Modern 3D tilt effects on news cards and a dynamic trending ticker.
- 🎙️ **Voice Search**: Search for news hands-free using integrated speech recognition.
- 🔄 **Infinite Loading**: Experience seamless content consumption with smart throttled scrolling.

---

## 🛠️ Technical Stack

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3.
- **Styling**: Modern CSS with custom properties (CSS Variables), Flexbox, and Grid.
- **Icons**: Font Awesome 6.
- **Typography**: Google Fonts (Outfit).
- **API**: [NewsData.io API](https://newsdata.io/) integration.

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge).
- An active API key from [NewsData.io](https://newsdata.io/register).

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/news-api-pro.git
   cd news-api-pro
   ```

2. **Set your API Key**:
   Open `script.js` and replace the `API_KEY` constant with your key:
   ```javascript
   const API_KEY = "YOUR_NEWSDATA_IO_KEY";
   ```

3. **Launch the App**:
   Simply open `index.html` in your browser or use a Live Server extension in VS Code.

## 🌟 Interactive Features Guide

- **Sidebar Toggle**: On mobile, click the ⚡ icon to slide out the category menu.
- **Card Tilt**: Hover over any news card on desktop to see the smooth 3D perspective effect.
- **Theme Switch**: Click the 🌙/☀️ icon in the top right to instantly toggle between Dark and Light modes.
- **Voice Search**: Tap the 🎤 icon, speak your query, and watch the results appear automatically.
